# Changelog

All notable changes to Lumora UI will be documented here.

This project uses Changesets for package changelogs and versioning.

## 0.1.0 - Initial release candidate

- Added `@lumora-ui/core` Tailwind v4 plugin with the `lm-*` component class contract.
- Added `@lumora-ui/themes` with 20+ WCAG-checked themes and required token exports.
- Added beta React and Vue wrapper packages.
- Added documentation site, accessibility guidance, enterprise workflow patterns, and visual regression tests.
